
# 🌍 InAmigos Foundation Website

A modern, responsive website built for **InAmigos Foundation**, a Section 8 non-profit organization working across India to create impact in education, food security, women empowerment, animal welfare, and environmental sustainability.

---

## 🚀 About the Project

This project is a **fully responsive NGO website** designed using HTML, CSS, and JavaScript. It showcases the foundation’s mission, projects, team, and impact in a visually appealing and user-friendly way.

The website highlights:

* Social initiatives
* Volunteer opportunities
* Donation links
* Real-world impact metrics
